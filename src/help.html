1. Overview
RemoteBorne Manager is a desktop application to manage IOTECHA charging stations via SSH. It provides:
  • Remote file access
  • Configuration editing
  • Energy Manager control
  • Service restart
  • Info & diagnostics
2. Installation
Requirements:
  • Windows 10+
  • Python 3.8+
  • plink.exe & pscp.exe in tools/
Installation: 
pip install ttkbootstrap
pip install reportlab
pip install pillow
3. File Management
Actions:
  • Download
  • Print as PDF
  • Edit & upload
  • Copy GridCodes
4. Energy Manager
Supports:
  • P / Q mode
  • CosPhi mode
  • Service restart
Commands are executed via: 
/usr/local/bin/EnergyManagerTestingTool
5. SSH System
Features:
  • Automatic reconnection
  • Dynamic target update
  • Threaded execution
  • SCP file transfer
Reconnect strategy: 
1s, 3s, 6s, 10s…
6. Network Configuration
RemoteBorne Manager stores settings in: 
config/config.ini
Changing host forces auto reconnect.
7. Debug Tools
  • Live logs window
  • Save logs
  • Clear logs
8. Performance Tips
  • Prefer LAN over Wi-Fi
  • Local SSD improves SCP speed
  • Disable antivirus for folder
9. Known Limitations
  • No SSH key authentication
  • No multi-device management
  • No realtime charts
10. Commands Reference
Energy commands: 
--power 

--reactive-power 
--cosphi
Compute CosPhi: 
Q = |P| * tan(acos(cosphi))
